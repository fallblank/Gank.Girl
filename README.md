# Gank.Girl
一个关于妹子图的应用，图片数据来源于gank.io
##API

##用到的知识点
1.SwipeRefreshLayout的使用实现下来刷新
2.CardView的使用实现圆角
3.LruCache实现图片的内存缓存
4.<a href="https://github.com/square/okhttp">OkHttp</a>的使用获取数据
5.图片的异步加载与压缩展示
